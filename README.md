# braingoat

Braingoat is a programming language which is made to develop brainfuck programms with a higher level of abstraction.

## Examples

### Int

```js
Int a = 1
Int b = 2
Int c = 3

Int sum = (b + (c - a))
print(sum)

Int power = (b ^ (a + c))
print(power)
```

### IntList

```js
IntList<4> myList = [42,43,44,45]

print(myList[0])
print(myList[1])
print(myList[2])
print(myList[3])
```
